# Monitoramento de Saúde – Aplicativo Android

## 1. Descrição do Problema
O aumento de doenças crônicas, como diabetes, obesidade e problemas cardiovasculares, torna essencial o acompanhamento regular de indicadores de saúde. No entanto, muitas pessoas — especialmente idosos — não possuem ferramentas simples para registrar e acompanhar sua evolução diária. Isso dificulta a identificação de padrões e o controle adequado da saúde.

Este aplicativo foi desenvolvido para oferecer uma solução prática e acessível que permita registrar e visualizar dados essenciais diretamente no dispositivo móvel.

---

## 2. Justificativa
O acompanhamento contínuo de fatores como batimentos cardíacos, peso e glicemia é fundamental para prevenir complicações e melhorar a qualidade de vida. O aplicativo contribui diretamente para o autocuidado, tornando esses registros simples e rápidos.

Este projeto está alinhado ao ODS 3 – Saúde e Bem-Estar, Meta 3.4, que visa reduzir doenças não transmissíveis por meio de prevenção e monitoramento.

---

## 3. Público-Alvo
O aplicativo foi desenvolvido para:
- Idosos que precisam monitorar sua saúde com frequência;
- Pessoas com diabetes, hipertensão ou outras condições crônicas;
- Usuários interessados em acompanhar indicadores básicos de saúde e sua evolução ao longo do tempo.

---

## 4. Objetivos do Aplicativo

### Objetivo Geral
Desenvolver um aplicativo Android simples e funcional para registrar, armazenar e consultar indicadores básicos de saúde.

### Objetivos Específicos
- Registrar batimentos cardíacos, peso corporal e glicemia;
- Permitir cadastro e login do usuário;
- Exibir histórico completo dos registros;
- Armazenar dados localmente com Room Database;
- Fornecer interface intuitiva e acessível construída com Jetpack Compose.

---

## 5. Tipo de Aplicação
O projeto é um **Aplicativo Nativo Android**, desenvolvido utilizando:
- Kotlin  
- Jetpack Compose  
- Room Database  
- Navigation Compose  

---

## 6. Instruções de Instalação e Uso

### 1. Clonar o Repositório
Para obter o código-fonte do aplicativo, execute o comando abaixo no terminal:
  - Download do zip dos arquivos
  - Extrair para uma pasta

### 2. Abrir o Projeto no Android Studio
1. Inicie o Android Studio.  
2. No menu principal, selecione **File > Open**.  
3. Navegue até a pasta onde o repositório foi clonado e selecione o diretório do projeto.  
4. Aguarde a conclusão da sincronização do Gradle, necessária para carregar todas as dependências.

---

### 3. Executar o Aplicativo
1. Conecte um dispositivo Android físico ao computador ou inicialize um emulador pelo Android Studio.  
2. No Android Studio, clique no botão **Run** (Executar).  
3. Selecione o dispositivo desejado para instalação.  
4. O aplicativo será instalado automaticamente e iniciado no dispositivo selecionado.

---

### 4. Uso do Aplicativo
1. Após a instalação, abra o aplicativo.  
2. Na tela inicial, realize o cadastro criando uma nova conta.  
3. Faça login utilizando as credenciais cadastradas.  
4. Registre novas medições, como batimentos cardíacos, peso corporal ou glicemia.  
5. Acesse o menu de histórico para visualizar todas as medições armazenadas localmente no dispositivo.

---

## 7. Requisitos do Sistema

### Sistema Operacional
- Dispositivos com **Android 7.0 (API 24)** ou superior.

### Ferramentas Necessárias para Desenvolvimento
- **Android Studio Hedgehog** ou **Android Studio Otter**.  
- **JDK** compatível com a versão utilizada pelo Android Studio.

### Dependências Principais do Projeto
- Kotlin  
- Jetpack Compose  
- Room Database  
- Navigation Compose
